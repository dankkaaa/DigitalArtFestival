# Digital Art Festival

## 🛠 Технологии

- **Backend**: Django 4 / 5, PostgreSQL
- **Frontend**: Bootstrap 5, HTML5, CSS3 (пиксельная стилистика), JavaScript (vanilla)

📁 Структура проекта
digital_fest/ — основной каталог проекта Django

gallery/ — приложение для галереи (модели работ, лайки)

program/ — приложение для программы событий

gen/ — generative stage (страница с интерактивом)

static/ и templates/ — статика и шаблоны

docker-compose.yml — конфигурация Docker

Dockerfile — образ приложения
