## ✨ Почему мы победим?

Мы создали не просто очередной сайт с карточками, а **полноценный интерактивный опыт**, который вовлекает пользователя с первой секунды. Каждая деталь продумана до пикселя, каждая анимация работает на атмосферу.

### 🚀 Ключевые особенности

| | |
|---|---|
| **🎬 Кинематографичное открытие** | При входе на главную цветные полосы разъезжаются, а текст печатается как на пишущей машинке — сразу видно, что сайт живой. |

| **🎭 Регистрация и события** | Пользователи могут записываться на лекции и перформансы, видеть статус (завершено / активно) и отменять запись. |

## 🎯 Как это работает (для жюри)

Мы не просто накидали шаблон, а продумали каждый сценарий:

- **Главная** — первое впечатление. Анимация раскрытия задаёт тон, а печатающийся текст приковывает внимание. Две крупные кнопки ведут в самые важные разделы.
- **Галерея** — сердце проекта. Карточки можно открыть в лайтбоксе или на отдельной странице.
- **Программа** — динамическая лента событий. Если событие прошло — помечается «Завершено», запись блокируется. Пользователь видит свой статус (записан / не записан).
- **Регистрация** — кастомная форма в стиле сайта, без стандартной скуки.
- **Дорожная карта** — показывает амбиции проекта до 2030 года. Прогресс-бар не просто украшение, он отражает реальный скролл и служит навигацией.
